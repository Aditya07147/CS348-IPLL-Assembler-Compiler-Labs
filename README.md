# IPLL Lab Assignments

This repository collects four programming-language lab assignments in a clean, assignment-wise layout.

## At A Glance

- [Assignment1](Assignment1) - NASM x86 assembly programs, including arithmetic and file-based tasks
- [Assignment2](Assignment2) - One-pass and two-pass SIC assembler implementations in C
- [Assignment3](Assignment3) - Flex-based lexical analyzer for nanoC
- [Assignment4](Assignment4) - Flex/Bison syntax analyzer for nanoC

## Why This Structure Works

Yes, assignment-wise grouping is a good way to present coursework on GitHub and in a CV. It keeps the scope obvious, makes each project easy to inspect, and lets a reviewer compare the assignments without digging through unrelated files.

## Portfolio Notes

- Filenames were normalized to remove duplicate archive-style suffixes.
- Each assignment stays self-contained with its own build notes.
- The repository was validated on Linux via WSL, so the projects are not Windows-only.

## Suggested CV Description

Built and organized four programming-language lab projects covering NASM assembly, SIC assembler construction, and nanoC front-end tooling with Flex and Bison; cleaned the repository structure for GitHub presentation and verified builds on Linux.
